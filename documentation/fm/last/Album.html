addTags()method
public function addTags(artist:String, album:String, tags:String):void

Add tags to a specific album.

Parameters
artist:String — the name of the artist.
 
album:String — the name of the album.
 
tags:String — a comma delimited list of user supplied tags to add to the album. Last.fm web services allow a maximum of 10 (currently only 1 allowed).

Example
Example of method in use: 
 
var _album:Album = new Album();

_album.addTags("Autechre", "Confield", "glitch");
_album.addEventListener(Album.ADD_TAGS, addTagsHandler);

function addTagsHandler(event:Event):void
{
    trace("tags added");
}

getInfo()method 
public function getInfo(artist:String = "", album:String = "", mbid:String = "", lang:String = ""):void

Get the metadata for a specific album using all or a combination of the artist name, album name or musicbrainz id.

Parameters
artist:String (default = "") — the name of the artist.
 
album:String (default = "") — the name of the album.
 
mbid:String (default = "") — the musicbrainz id for the album.
 
lang:String (default = "") — the language to return the biography in, expressed as an ISO 639 alpha-2 code.

Example
Example of method in use: 
 
var _album:Album = new Album();

_album.getInfo("Autechre", "Amber");
_album.addEventListener(Album.GET_INFO, getInfoHandler);

function getInfoHandler(event:Event):void
{
    trace(_album.xml);
    var album:XMLList = _album.xml.album;
    trace("Album name: " + album.name);
    trace("Album artist: " + album.artist);
    trace("Album id: " + album.id);
    trace("Album mbid: " + album.mbid);
    trace("Album url: " + album.url);
    trace("Album release date: " + album.releasedate);
    for each(var image:XML in album.image)
    {
        trace("Image size: " + image.@size + " and URL: " + image);
    }
    trace("Album listeners: " + album.listeners);
    trace("Album playcount: " + album.playcount);
    trace("Album toptags: " + album.toptags);
}

search()method 
public function search(album:String, limit:uint = 30, page:uint = 1):void

Search for an album by the album name.

Parameters
album:String — the name of the album.
 
limit:uint (default = 30) — the number of search results to be returned. Default (maximum) is 30.
 
page:uint (default = 1) — the number of the results page to view. Default is 1.

Example
Example of method in use: 
 
var _album:Album = new Album();

_album.search("Amber");
_album.addEventListener(Album.SEARCH, searchHandler);

function searchHandler(event:Event):void
{
    trace(_album.xml.results.albummatches);
    var albums:XMLList = _album.xml.results.albummatches.album;

    for each (var album:XML in albums)
    {
        trace("Album name: " + album.name);
        trace("Album artist: " + album.artist);
        trace("Album id: " + album.id);
        trace("Album url: " + album.url);
        for each(var image:XML in album.image)
        {
            trace("Image size: " + image.@size + " and URL: " + image);
        }
        trace("Album streamable: " + album.streamable + "\n");
    }
}
