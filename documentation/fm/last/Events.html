public function share(event:Number, recipient:String, message:String):void

Share an event with Last.fm users or other friends.

Parameters
event:Number — the id of the event.
 
recipient:String — a comma delimited list of email addresses or Last.fm usernames. The Last.fm web services allow a maximum of 10 (currently only 1 allowed).
 
message:String — the message to send to the friends.
