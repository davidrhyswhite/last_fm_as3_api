getEvents()method
public function getEvents(location:String = "", lat:Number = 0, long:Number = 0, page:uint = 1, distance:Number = 20):void

Get a list events based on location (city or country), if no location is specified a list of nearby events is returned.

Parameters
location:String (default = "") — the location of the events returned, use either a country or city name.
 
lat:Number (default = 0) — the latitude of the events.
 
long:Number (default = 0) — the longditude of the events.
 
page:uint (default = 1) — the number of the results page to view. Default is 1.
 
distance:Number (default = 20) — the number of friends returned. Default is 20.
